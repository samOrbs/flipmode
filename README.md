# Flipmode - BJJ Training Companion for Obsidian

A powerful Obsidian plugin for BJJ athletes to document training, submit research queries, and receive personalized coaching insights.

## Features

- **Training Session Logging** - Document your rolls with structured data
- **Research Queries** - Submit technique questions to your analyst
- **Graph Sync** - Your training data syncs to your analyst's system
- **Discord Integration** - Connect via Discord for authentication

## Installation

### Via BRAT (Recommended)

1. Install [BRAT](https://github.com/TfTHacker/obsidian42-brat) from Obsidian Community Plugins
2. Open BRAT settings
3. Click "Add Beta Plugin"
4. Enter: `samOrbs/flipmode-obsidian-plugin`
5. Enable the plugin in Community Plugins

### Manual Installation

1. Download `main.js`, `manifest.json`, and `styles.css` from the latest release
2. Create folder: `YOUR_VAULT/.obsidian/plugins/flipmode/`
3. Copy the 3 files into that folder
4. Reload Obsidian
5. Enable "Flipmode" in Settings > Community Plugins

## Setup

1. Open plugin settings
2. Set mode to "Remote (Athlete)"
3. Click "Connect with Discord"
4. Complete authentication
5. Your token will be saved automatically

## Usage

### Log Training Sessions
- Use the ribbon icon or command palette
- Fill in training details (techniques, rolls, notes)
- Sessions sync automatically

### Submit Research Queries
- Build your query in a conversation note
- Click "Send to Analyst"
- Check back for results
